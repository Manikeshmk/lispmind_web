# 🧠 LispMind Web

AI Chatbot in your browser. Built with **ClojureScript (Lisp)**.

## Quick Start

```bash
npm install
npm run watch
```

Visit: **http://localhost:8080**

## Features

- 💬 Remembers your name
- 😊 Detects emotions
- 🔍 Symptom diagnosis
- 📚 Answers about Lisp & AI
- 100% client-side

## Try It

```
hello
my name is Alice
what is my name
I am sad
I have fever and cough
what is lisp
```

## Build

```bash
npm run build
```

## Tech

- ClojureScript (80%+)
- shadow-cljs
- HTML5 + CSS3
